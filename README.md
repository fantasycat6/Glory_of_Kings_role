# 王者荣耀英雄统计网站

一个基于 Node.js + Express + SQLite 的王者荣耀英雄收集统计网站。

## ✨ 功能特性

### 用户功能
- 🔐 用户注册/登录（JWT + Cookie 持久化）
- 👤 个人中心（修改资料、密码）
- 🎮 多账号管理（QQ/微信区，安卓/苹果设备）
- 🗺️ 自定义区服（支持任意区服名称）
- 🦸 英雄收集标记
- 📊 实时统计展示（总数、已收集、百分比、职业分布）
- 🔍 英雄搜索和职业筛选
- ✨ 批量操作（多选、框选、全选）

### 管理后台
- 👥 用户管理（增删改、密码重置）
- 🦸 英雄管理（增删改、分页、职业筛选、默认英雄设置）
- 💾 数据备份（导出、导入、恢复、删除）
- 📈 系统概览

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0
- npm >= 8.0

### 安装依赖
```bash
npm install
```

### 启动服务
```bash
# 开发模式
npm start

# 或使用 node
node app.js
```

访问 http://localhost:3000

### 默认管理员账号
- 用户名: `admin`
- 密码: `admin123`

## 🛠️ 技术栈

- **后端**: Node.js + Express
- **模板引擎**: EJS
- **数据库**: SQLite3
- **前端**: Bootstrap 5 + Bootstrap Icons
- **认证**: bcryptjs + jsonwebtoken + cookie-parser

## 📝 数据说明

### 英雄数据
- 总计 **130** 位英雄（包含5个元流之子）
- 职业分布：
  - 坦克: 30位
  - 战士: 43位
  - 刺客: 24位
  - 法师: 45位
  - 射手: 20位
  - 辅助: 22位

### 默认英雄（新账号初始拥有）
- 亚瑟、妲己、后羿、安琪拉
- 元流之子·坦克、元流之子·刺客、元流之子·法师、元流之子·射手、元流之子·辅助

## 💾 数据备份

支持多种备份操作：
- **创建备份**: 在服务器本地创建备份
- **导入备份**: 上传JSON文件恢复数据
- **恢复备份**: 从已有备份恢复
- **下载备份**: 下载备份文件到本地
- **删除备份**: 删除服务器上的备份

支持合并模式和覆盖模式导入。

## 🔒 安全说明

- 用户密码使用 bcrypt 加密存储
- JWT Token 存储在 Cookie 中，有效期7天
- 管理员权限验证
- SQL 注入防护（参数化查询）
